The **Charts** block lets you design stylish, customizable WordPress buttons with layout controls, pre-made styles, and integrated icons.

== Description ==

Chart Blocks for Gutenberg is a plugin that adds customizable chart and graph blocks to the WordPress block editor. With this plugin, users can easily embed interactive charts (bar, line, pie, etc.) into their posts or pages to display data in a visually appealing format.

== Accessing the Source Code ==

You can find the source code for this plugin on GitHub:
[GitHub Repository Link] - https://github.com/shamim496/zolo-charts.git

**Key Features:**

- Multiple Chart Types: Choose from bar charts, line charts, pie charts, and more.
- Fully Customizable: Customize chart colors, titles, axes, labels, and more.
- Responsive Design: Charts automatically adjust to fit any screen size.
- Data Inputs: Easily input data manually or upload CSV files for automatic chart generation.
- Interactive Charts: Hover effects, tooltips, and more.
- Live Data Integration: Integrate external APIs for live data charts (advanced users).
- Export Options: Allow users to download charts as images (PNG, SVG).

== Installation ==

= Plugin Installation Method: =

1. Go to the WordPress dashboard
2. Got to the **Plugins** > **Add New**
3. Type **"Charts"** in the search box
4. Click on **"Install"** button
5. After install the plugin **Activate** the plugin

= Installation via Zip file: =

1. Download Charts block **plugin zip** file from WordPress
2. Go to site **Dashboard** > **Plugins** > **Add New** > **Upload Plugins**
3. Select the Charts zip file and proceed to install
4. Activate Charts.

== Frequently Asked Questions ==

= How to Use? =

1. **Add the Block**:
   - Open the WordPress Block Editor (Gutenberg).
   - Click the "+" button to add a new block and search for "Charts."
   - Select and add the block to your post or page.

2. **Customize the Button**:
   - **Text**: Click on the button text and type your desired label (e.g., "Learn More").
   - **URL**: Add a URL to link the button to a specific page or site.
   - **Background Color**: Use the color picker in the block settings sidebar to select a background color.
   - **Outline Style**: Toggle the "Outline" option in the settings to create a border-only button.

3. **Preview Your Button**:
   - Use the preview option to see how the button looks on your website.

4. **Save Changes**:
   - Save or update your post/page to make the button live on your site.
