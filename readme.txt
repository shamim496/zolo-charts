== Installation ==

= Plugin Installation Method: =

1. Go to the WordPress dashboard
2. Got to the **Plugins** > **Add New**
3. Type **"Charts"** in the search box
4. Click on **"Install"** button
5. After install the plugin **Activate** the plugin

= Installation via Zip file: =

1. Download Charts block **plugin zip** file from WordPress
2. Go to site **Dashboard** > **Plugins** > **Add New** > **Upload Plugins**
3. Select the Charts zip file and proceed to install
4. Activate Charts.

== Frequently Asked Questions ==

= How to Use? =

1. **Add the Block**:
   - Open the WordPress Block Editor (Gutenberg).
   - Click the "+" button to add a new block and search for "Charts."
   - Select and add the block to your post or page.

2. **Customize the Button**:
   - **Text**: Click on the button text and type your desired label (e.g., "Learn More").
   - **URL**: Add a URL to link the button to a specific page or site.
   - **Background Color**: Use the color picker in the block settings sidebar to select a background color.
   - **Outline Style**: Toggle the "Outline" option in the settings to create a border-only button.

3. **Preview Your Button**:
   - Use the preview option to see how the button looks on your website.

4. **Save Changes**:
   - Save or update your post/page to make the button live on your site.

= What features does the plugin offer? =
Charts offers a range of features designed to enhance your web design experience. These includes customizable blocks, layouts, patterns,  drag-and-drop interface, advanced styling options, flexibility, responsiveness and more.

= Does Charts work with all WordPress themes? =
Most certainly all WordPress themes are compatible with the Zolo Blocks framework as the plugin is built for maximum compatibility with WordPress itself.
